Taraweeh Sajdah Checker

Toggle input type (Page range vs Surah:Ayah range). This checks whether a sajdah occurs in tonight’s recitation, and estimates which rak‘ah it will land in (8 and/or 20) based on how far into the entered range it occurs (estimate).
Format: surah:ayah-surah:ayah
Format: start-end or a single page number.
Some madhahib differ; this toggle only controls whether 22:77 is included.
It’s always an estimate (ayah lengths/pace vary).
Offline dataset: sajdah locations + page numbers (604-page mushaf) from AlQuran Cloud sajdah list. :contentReference[oaicite:1]{index=1}
