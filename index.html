Taraweeh Sajdah Checker

Toggle input type (Page range vs Surah:Ayah range). This tool checks whether a sajdah (tilawah prostration) occurs in tonight’s recitation, and estimates which rak‘ah it will land in (8 or 20) based on your entered range (an estimate).
Format: surah:ayah-surah:ayah
Format: start-end or a single page number.
If you pick “14-list”, this only removes 22:77 from the API’s list.
Either way is an estimate (ayah lengths vary).
Data source: AlQuran Cloud API. Cached in your browser for faster repeat checks.
